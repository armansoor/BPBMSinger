About this fan project

This is a fan-made generator inspired by BLACKPINK and BABYMONSTER. It runs fully in your browser: lyrics generation, live TTS playback (browser voices), and downloadable musical renders using WebAudio + MediaRecorder. No servers, no tokens, no account.

Copyright: © 2025 Abdur Rahman Mansoor — github.com/armansoor

Live TTS uses your browser's available voices (English/Korean). Downloadable audio is produced by an internal synth so the file is available across browsers.

Fan project — not affiliated with YG Entertainment
